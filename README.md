# lit-webpack-starter

## Setup dev environment

For npm:

```bash
npm i
```

For yarn:

```bash
yarn
```

## Bundle sources

To bundle your sources to JavaScript:

```bash
npm run build
```

or:

```bash
yarn build
```

## Running in dev

To serve static files in development:

```bash
npm run serve
```

or:

```bash
yarn serve
```

## Linting + Formatting

This template includes ESLint + Prettier.

To lint your sources:

```bash
npm run lint
```

or:

```bash
yarn lint
```

To format your sources:

```bash
npm run format
```

or:

```bash
yarn format
```
